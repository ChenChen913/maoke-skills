# 舌象自检指南（图文说明）

## 观察前准备

### 环境要求
- **光线**：自然光最佳（白天靠窗位置）
- **时间**：早晨起床后、刷牙前（避免食物染色）
- **姿势**：坐直，对着镜子
- **工具**：镜子（手持镜或浴室镜）

### 注意事项
- 观察前不要吃东西或喝水
- 不要用力伸舌头（自然伸出即可）
- 观察时间约 10-15 秒，不要过长
- 如刚喝过咖啡/茶/有色饮料，需漱口后等待 30 分钟

## 观察步骤

### 第一步：舌体颜色

**观察方法：** 看舌头整体颜色

| 颜色 | 图示描述 | 体质提示 |
|------|---------|---------|
| 淡白 | 像褪色的粉红色，缺乏血色 | 气虚质、阳虚质 |
| 淡红 | 健康的粉红色 | 平和质（正常） |
| 红 | 明显的红色，像草莓色 | 阴虚质、湿热质 |
| 深红/暗红 | 深红色，甚至有点发紫 | 热证、血瘀质 |
| 暗紫 | 舌体发紫，或有紫色瘀点 | 血瘀质 |

---

### 第二步：舌体形态

**观察方法：** 看舌头的形状和大小

| 形态 | 图示描述 | 体质提示 |
|------|---------|---------|
| 偏胖 | 舌头伸出来感觉嘴巴装不下 | 痰湿质、阳虚质 |
| 偏瘦 | 舌头薄薄的，细细的 | 阴虚质 |
| 正常 | 大小适中 | 平和质（正常） |
| 齿痕 | 舌头边缘有波浪状的牙齿印 | 气虚质、痰湿质 |
| 裂纹 | 舌面上有裂纹，像干裂的土地 | 阴虚质、血虚 |

---

### 第三步：舌苔

**观察方法：** 看舌面上的"苔藓"状覆盖物

| 舌苔 | 图示描述 | 体质提示 |
|------|---------|---------|
| 薄白 | 一层薄薄的白色，舌体颜色隐约可见 | 平和质（正常） |
| 厚白 | 白色苔很厚，看不到舌体颜色 | 寒湿、痰湿质 |
| 黄苔 | 苔发黄，可能厚也可能薄 | 热证、湿热质 |
| 腻苔 | 苔看起来油腻腻的，黏黏的 | 痰湿质、湿热质 |
| 少苔/无苔 | 舌面很光滑，没什么苔 | 阴虚质 |
| 偏一侧 | 一边苔厚，一边苔薄 | 气机不畅 |

---

### 第四步：舌面湿润度

**观察方法：** 看舌面是否有水光感

| 湿润度 | 图示描述 | 体质提示 |
|--------|---------|---------|
| 偏干 | 舌面干燥，没有光泽 | 阴虚质、热证 |
| 正常 | 舌面湿润，有光泽 | 平和质（正常） |
| 水滑 | 舌面水汪汪的，像有水珠 | 阳虚质、痰湿质 |

## 常见舌象组合解读

### 气虚质舌象
```
舌体：淡白，偏胖
舌苔：薄白
特征：边缘有齿痕
湿润度：正常或偏湿
```

### 阳虚质舌象
```
舌体：淡白，胖大
舌苔：白苔（可能厚）
特征：边缘有齿痕
湿润度：水滑（像有水珠）
```

### 阴虚质舌象
```
舌体：红，偏瘦
舌苔：少苔或无苔
特征：舌面可能有裂纹
湿润度：偏干
```

### 痰湿质舌象
```
舌体：淡白，胖大
舌苔：白腻苔（厚且油腻）
特征：边缘有齿痕
湿润度：水滑
```

### 湿热质舌象
```
舌体：红
舌苔：黄腻苔（黄且油腻）
特征：可能有点刺
湿润度：正常或偏干
```

### 血瘀质舌象
```
舌体：暗紫或有紫色斑点
舌苔：薄白或薄黄
特征：舌面可能有瘀点
湿润度：正常
```

## 舌象变化追踪

### 改善信号（体质正在改善）
- ✅ 舌苔由厚变薄（痰湿在减轻）
- ✅ 齿痕减少（气虚在改善）
- ✅ 舌色由淡白变淡红（阳气在恢复）
- ✅ 苔腻感减轻（湿热/痰湿改善）

### 需要关注的信号
- ⚠️ 舌苔突然变黄（可能有炎症或上火）
- ⚠️ 舌体颜色变深红（阴虚加重或有热）
- ⚠️ 出现新的瘀点瘀斑（血瘀加重）
- ⚠️ 舌苔完全脱落（胃阴受损）
