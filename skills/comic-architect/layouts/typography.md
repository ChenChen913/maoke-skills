# Typography & Text Rendering

For models with text rendering capabilities (like Ideogram, Flux, Midjourney v6), typography is a key visual element.

## 1. Font Styles
- **Swiss Sans**: Helvetica-ish, bold, neutral, clean. (e.g., "Inter", "Roboto").
- **Serif Editorial**: Elegant, high-contrast, fashion magazine vibe. (e.g., "Didot", "Playfair Display").
- **Hand-Drawn**: Marker, pencil, organic, imperfect. (e.g., "Comic Sans" but good, "Amatic").
- **Pixel**: Blocky, retro gaming.
- **Monospace**: Code-like, typewriter, technical. (e.g., "Courier", "Fira Code").
- **Calligraphy**: Brush strokes, flowing, traditional.
- **Bubble/Graffiti**: Inflated, street art, overlapping.

## 2. Text Placement Strategies
- **Headline Hero**: Massive text overlaying the image (Magazine Cover style).
- **Labeling**: Small text with lines pointing to object parts (Scientific Diagram).
- **Speech Bubbles**: Text inside ovals/clouds (Comic style).
- **Floating UI**: Text inside semi-transparent cards/glass elements.
- **Integrated/Embedded**: Text that is part of the object (carved into wood, neon sign, icing on cake).

## 3. Text Effects
- **Neon Glow**: Outer glow, bright core.
- **Letterpress**: Pressed into the paper, inner shadow.
- **Gold Foil**: Metallic reflection.
- **Glitch**: Split RGB channels, sliced text.
- **3D Extrusion**: Thick block letters with perspective.
