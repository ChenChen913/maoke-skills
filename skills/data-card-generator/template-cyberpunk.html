CLAUDE CODE

核心命令 · 速查手册

01
CMD/btw · 旁路提问

执行任务时插入临时问题，不污染主上下文，回答后自动消除

02
CMD/rewind · 分层回退

双 Esc 触发，支持回退代码、对话或两者，可压缩上下文释放空间

03
CMD/model · 智能切换

规划用 Opus，执行用 Sonnet，在效果与额度之间动态平衡

04
CMD/branch · 会话分叉

克隆当前对话探索不同方案，原会话保持不变，择优采用

05
CMD/rc · 远程控制

生成 URL 供手机打开，双端实时同步，代码始终在本地运行
