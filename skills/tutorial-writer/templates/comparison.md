# 对比 / 选型类教程模板

适用于：帮助读者在多个相似的工具、框架或方案之间做选择。
例如：Vue vs React、PostgreSQL vs MySQL、Docker vs 虚拟机、各类 LLM API 的对比等。

---

## 写作要点

这类教程的读者处于决策状态，他们不需要深入学习某一个工具，而是需要足够的信息做出合理的选择。写作重点是：
- 客观，不强行推荐
- 比较维度统一（所有候选方案用相同维度衡量）
- 结论明确（在什么情况下选什么）

---

- 本文的目标：不是评出"最好的"，而是帮你找到"最适合你场景的"
