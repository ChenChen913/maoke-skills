# 安装 / 配置 / 部署类教程模板

适用于：核心目标是让读者把某个软件或服务跑起来，并完成基本配置。
例如：安装配置 OpenClaw、部署 Nginx、配置 SSH 免密登录、搭建本地开发环境等。

---

## 写作要点

这类教程的读者目标明确：跑通，不报错。写作重点是：
- 步骤完整，不跳跃
- 命令可以直接复制粘贴
- 每个关键步骤给出预期输出（让读者能自行判断是否成功）
- 常见报错必须覆盖

---
